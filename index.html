FAQ

Platform notes

How does importing work?

The prototype validates URL-shaped input and adds metadata locally. Production should use API validation, a queue, FFmpeg or Remotion rendering, temporary storage, and thumbnail generation.

What determines the ranking order?

User-defined order is the default. Move controls and list selection update the preview, timeline, and export sequence.

Where should production services run?

A practical setup is Next.js on Vercel, render workers on Render, Fly.io, or Railway, Redis for jobs, PostgreSQL for projects, and R2 or S3 for cached media.
